# 커밋 메시지 컨벤션

커밋 메시지의 헤더는 대괄호(`[]`)로 타입을 표시합니다.

```
[타입] 커밋 내용 요약
```

## 타입 목록
아래의 헤더 종류 외에는 절대 사용하지 않는다.
| 타입 | 설명 |
| --- | --- |
| `[fix]` | 버그 수정 |
| `[feat]` | 새로운 기능 추가 |
| `[docs]` | 문서 수정 |
| `[style]` | 코드 포맷팅, 세미콜론 누락 등 (기능 변경 없음) |
| `[refactor]` | 코드 리팩토링 |
| `[test]` | 테스트 코드 추가/수정 |
| `[chore]` | 빌드 설정, 패키지 매니저 설정 등 기타 변경 |

## 예시
```
[fix] 긴급 Bug Fix
```
