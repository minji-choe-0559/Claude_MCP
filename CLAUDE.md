# CLAUDE.md

이 저장소는 **Git/GitHub 사용법 실습**을 위한 프로젝트입니다. 실제 애플리케이션 코드를 작성하는 곳이 아닙니다.

## 프로젝트 목적
- 커밋, 브랜치, 푸시, 풀 리퀘스트, 머지, 충돌 해결 등 Git/GitHub 워크플로우를 연습하기 위한 용도입니다.
- 실습 내용은 `README.md`와 `docs/` 폴더 아래의 문서에 기록합니다.

## 작업 규칙
- 실습용 애플리케이션 코드, 설정 파일 등을 추가로 만들지 않습니다. `README.md`와 `docs/` 문서만 생성/수정합니다.
- Visual Studio가 자동 생성한 `.slnx`, `.vs` 등의 파일은 건드리지 않습니다.
- 커밋 메시지는 `docs/COMMIT_CONVENTION.md`에 정의된 컨벤션(`[타입] 요약` 형식)을 따릅니다.
- 브랜치 이름은 실습 목적에 맞게 간단하고 명확하게 작성합니다.
- git 관련 요청(커밋, 브랜치, 푸시 등)은 반드시 사용자에게 실행 전 확인을 받습니다.
